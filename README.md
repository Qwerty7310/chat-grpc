# chat-grpc

gRPC‑сервис чата на Go с поддержкой двунаправленного стриминга сообщений
между клиентами через сервер.

Проект демонстрирует использование **gRPC**, **Protocol Buffers** и
конкурентной обработки подключений в **Go** для реализации real‑time
обмена сообщениями.

------------------------------------------------------------------------

# Функциональность

-   подключение нескольких клиентов к серверу
-   обмен сообщениями в реальном времени
-   двунаправленный streaming RPC
-   широковещательная доставка сообщений всем подключённым клиентам
-   CLI‑клиент для взаимодействия с сервером

------------------------------------------------------------------------

# Стек технологий

-   Go
-   gRPC
-   Protocol Buffers
-   HTTP/2

------------------------------------------------------------------------

# Архитектура

Система реализована по модели **client‑server**.

Сервер:

-   принимает gRPC соединения клиентов
-   обрабатывает входящие сообщения
-   рассылает сообщения всем подключённым клиентам

Клиенты:

-   устанавливают gRPC stream с сервером
-   отправляют сообщения
-   получают сообщения от других клиентов через сервер

# gRPC сервис

Описание сервиса выполняется через Protocol Buffers.

Пример структуры:

``` proto
syntax = "proto3";

package chat;

service ChatService {
  rpc SendMessage(Message) returns (Empty);
  rpc StreamMessages(Empty) returns (stream Message);
}

message Message {
  string user = 1;
  string text = 2;
}

message Empty {}
```

# Установка

Клонировать репозиторий:

    git clone https://github.com/Qwerty7310/chat-grpc.git
    cd chat-grpc

Установить зависимости:

    go mod tidy

Сгенерировать gRPC код (если требуется):

    protoc --go_out=. --go-grpc_out=. proto/chat.proto

------------------------------------------------------------------------

# Запуск

Запуск сервера:

    go run server/main.go

Сервер запускается на:

    localhost:50051

Запуск клиента:

    go run client/main.go

Можно запускать несколько клиентов одновременно для обмена сообщениями.
